# Sistema Boutique Maquillaje - Evaluación 3

Sistema POO con API simulada, encriptación y base de datos MySQL.

## Requisitos
- Python 3.8+
- XAMPP (MySQL)
- Dependencias en requirements.txt

## Instalación
1. Activar MySQL en XAMPP
2. Crear BD: `boutique_api`
3. Ejecutar scripts SQL en orden
4. `pip install -r requirements.txt`
5. `python sistema_boutique.py`

## Estructura
- auxiliares/ → Configuración
- datos/ → Conexión BD y SQL
- modelos/ → Modelos SQLAlchemy
